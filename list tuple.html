השוואה בין Tuples לרשימות בפייתון

ההבדלים, השימושים והיתרונות

רשימות (Lists): [ ]
טאפלים (Tuples): ( )
ההבדל העיקרי: רשימות ניתנות לשינוי (mutable), בעוד שטאפלים אינם ניתנים לשינוי (immutable)
יצירת טאפלים ורשימות
רשימות (Lists)
my_list = [1, 2, 3, 4, 5] print(my_list)
[1, 2, 3, 4, 5]
mixed_list = [1, "שלום", 3.14, True] print(mixed_list)
[1, 'שלום', 3.14, True]
single_item_list = [42] print(type(single_item_list))
<class 'list'>
טאפלים (Tuples)
my_tuple = (1, 2, 3, 4, 5) print(my_tuple)
(1, 2, 3, 4, 5)
mixed_tuple = (1, "שלום", 3.14, True) print(mixed_tuple)
(1, 'שלום', 3.14, True)
single_item_tuple = (42,) # שים לב לפסיק! print(type(single_item_tuple))
<class 'tuple'>
חשוב! כדי ליצור טאפל עם ערך אחד, חובה להוסיף פסיק אחרי הערך:
single_tuple = (42,) # זהו טאפל just_number = (42) # זהו סתם מספר בסוגריים print(type(single_tuple)) print(type(just_number))
<class 'tuple'> <class 'int'>
פעולות הוספה
1. הוספת איבר בסוף (append)
ברשימה
a = [10, 20, 30] a.append(40) print(a)
[10, 20, 30, 40]
בטאפל
a = (10, 20, 30) # אין מתודת append בטאפל # דרך חלופית - יצירת טאפל חדש: b = a + (40,) # שים לב לפסיק print(b)
(10, 20, 30, 40)
2. הרחבת האוסף (extend)
ברשימה
a = [10, 20, 30] a.extend([40, 50]) print(a)
[10, 20, 30, 40, 50]
בטאפל
a = (10, 20, 30) # אין מתודת extend בטאפל # דרך חלופית - יצירת טאפל חדש: b = a + (40, 50) print(b)
(10, 20, 30, 40, 50)
3. הוספת איבר באינדקס ספציפי (insert)
ברשימה
a = [10, 20, 40] a.insert(2, 30) # הוספת 30 באינדקס 2 print(a)
[10, 20, 30, 40]
בטאפל
a = (10, 20, 40) # אין מתודת insert בטאפל # דרך חלופית - יצירת טאפל חדש: b = a[:2] + (30,) + a[2:] print(b)
(10, 20, 30, 40)
פעולות חיפוש ומידע
1. ספירת הופעות של ערך (count)
ברשימה
a = [10, 20, 30, 20] print(a.count(20))
2
בטאפל
a = (10, 20, 30, 20) print(a.count(20))
2
2. מציאת אינדקס של ערך (index)
ברשימה
a = [10, 20, 30, 20] print(a.index(20))
1
בטאפל
a = (10, 20, 30, 20) print(a.index(20))
1
שים לב: המתודות count() ו-index() פועלות באופן זהה בטאפלים וברשימות. אלו המתודות היחידות הקיימות בטאפלים, והן לא משנות את מצב האובייקט אלא רק מחזירות מידע. גם ברשימות פעולות אלו רק מחזירות מידע ולא משנות את הרשימה.
פעולות הסרה והיפוך
1. היפוך סדר האיברים (reverse)
ברשימה
a = [10, 20, 30, 40] a.reverse() print(a)
[40, 30, 20, 10]
בטאפל
a = (10, 20, 30, 40) # אין מתודת reverse בטאפל # דרך חלופית - יצירת טאפל חדש: b = a[::-1] # או tuple(reversed(a)) print(b)
(40, 30, 20, 10)
2. הסרת ערך מהאוסף (remove)
ברשימה
a = [10, 20, 30, 10] a.remove(10) # מסיר את ההופעה הראשונה print(a)
[20, 30, 10]
בטאפל
a = (10, 20, 30, 10) # אין מתודת remove בטאפל # דרך חלופית - יצירת טאפל חדש: index = a.index(10) b = a[:index] + a[index+1:] print(b)
(20, 30, 10)
3. הסרת איבר לפי אינדקס (pop)
ברשימה
a = [10, 10, 20, 30] a.pop(0) # מסיר את האיבר באינדקס 0 print(a) a.pop() # מסיר את האיבר האחרון print(a)
[10, 20, 30] [10, 20]
בטאפל
a = (10, 10, 20, 30) # אין מתודת pop בטאפל # דרך חלופית - יצירת טאפלים חדשים: b = a[1:] # ללא האיבר הראשון print(b) c = a[:-1] # ללא האיבר האחרון print(c)
(10, 20, 30) (10, 10, 20)
פעולות מיון
1. מיון במקום (sort)
ברשימה
a = [20, 40, 30, 10] a.sort() # מיון מהקטן לגדול print(a)
[10, 20, 30, 40]
a = [20, 40, 30, 10] a.sort(reverse=True) # מיון מהגדול לקטן print(a)
[40, 30, 20, 10]
בטאפל
a = (20, 40, 30, 10) # אין מתודת sort בטאפל # דרך חלופית - שימוש בפונקציית sorted: b = tuple(sorted(a)) # מהקטן לגדול print(b)
(10, 20, 30, 40)
a = (20, 40, 30, 10) c = tuple(sorted(a, reverse=True)) # מהגדול לקטן print(c)
(40, 30, 20, 10)
2. מיון ללא שינוי האוסף המקורי (sorted)
ברשימה
a = [30, 10, 40, 20] result = sorted(a) print(result) # רשימה חדשה ממוינת print(a) # הרשימה המקורית לא השתנתה
[10, 20, 30, 40] [30, 10, 40, 20]
בטאפל
a = (30, 10, 40, 20) result = sorted(a) print(result) # רשימה ממוינת (לא טאפל!) print(a) # הטאפל המקורי לא השתנה
[10, 20, 30, 40] (30, 10, 40, 20)
הבדל חשוב:
  • המתודה sort() ברשימה משנה את הרשימה עצמה ("במקום").
  • הפונקציה sorted() עובדת על כל אוסף (רשימה, טאפל וכו') ומחזירה רשימה חדשה ממוינת, בלי לשנות את האוסף המקורי.
  • כדי לקבל טאפל ממוין, יש להשתמש ב-tuple(sorted(my_tuple)).

הוספת איבר בסוף a.append(40) ✓ משנה את הרשימה ✗ לא קיים כן (ברשימה)
הרחבת האוסף a.extend([40, 50]) ✓ משנה את הרשימה ✗ לא קיים כן (ברשימה)

ספירת הופעות a.count(20) ✓ מחזירה ערך ✓ מחזירה ערך לא
מציאת אינדקס a.index(20) ✓ מחזירה ערך ✓ מחזירה ערך לא
היפוך סדר ()a.reverse ✓ משנה את הרשימה ✗ לא קיים כן (ברשימה)
הסרת ערך a.remove(10) ✓ משנה את הרשימה ✗ לא קיים כן (ברשימה)
הסרה לפי אינדקס ()a.pop ✓ משנה את הרשימה ✗ לא קיים כן (ברשימה)
מיון במקום ()a.sort ✓ משנה את הרשימה ✗ לא קיים כן (ברשימה)
מיון לאוסף חדש sorted(a) ✓ מחזירה רשימה חדשה ✓ מחזירה רשימה חדשה לא
מסקנה: הפעולות היחידות שפועלות באופן זהה על טאפלים ורשימות (מסומנות ברקע ירקרק בטבלה) הן ()count ()index ופונקציית ()sorted. שים לב ש-()sorted מחזירה רשימה ולא משנה את האוסף המקורי.

מתי להשתמש בטאפל?
  • כשהנתונים צריכים להישאר קבועים
  • כמפתחות במילונים
  • להחזרת מספר ערכים מפונקציה
  • להגנה על נתונים מפני שינויים
מתי להשתמש ברשימה?
  • כשהנתונים צריכים להשתנות
  • כשצריך להוסיף או להסיר פריטים
  • כשצריך למיין או להפוך את הסדר במקום
  • כשאוספים נתונים באופן דינמי
